GenAI - API Proxy Service

Please be aware that although GenAI and its affiliated organization - GitData - do not gather user information, the server providers for GenAI API proxy service and the Generative AI service providers may engage in such data collection. Furthermore, the proxy service cannot guarantee a consistent connection speed. Users are strongly encouraged to utilize this service with caution and at their own discretion.
