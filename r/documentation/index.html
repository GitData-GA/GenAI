GenAI - R Package "GenAI" Documentation

Last update: 2024/01/18

Utilizing 'Generative Artificial Intelligence' models like 'GPT-4' and 'Gemini Pro' as coding and writing assistants for 'R' users. Through these models, 'GenAI' offers a variety of functions, encompassing text generation, code optimization, natural language processing, chat, and image interpretation. The goal is to aid 'R' users in streamlining laborious coding and language processing tasks.

Prerequisites

Prior to utilizing the GenAI package, several prerequisites must be met.

  1. Ensure that you possess an eligible device equipped with R.

  2. Access to the internet is essential to generate text or engage in chat through GenAI.

  3. Obtain an API key from the selected Generative AI service provider. GenAI currently supports Generative AI models from both Google and OpenAI.

    • To acquire an API key for Google's models, refer to: Get an API key
    • To acquire an API key for OpenAI's models, refer to: Account setup

Installation

1. Install the package from The Comprehensive R Archive Network (CRAN).

install.packages("GenAI")

2. Install the package from GenAI official website.

install.packages("https://genai.gd.edu.kg/release/R/GenAI_latest.tar.gz", repos=NULL, method="libcurl")

Setup

The "Setup" section provides functions to explore available Generative AI models. The available.models function fetches information about supported models, while the connect.genai function establishes a connection to a specific model using key parameters like service provider, model, version, and API key.

available.models

Get supported Generative AI models

Usage
available.models()
Value

If successful, the function returns a list containing Generative AI service providers and their corresponding models. If the function encounters an error, it will halt execution and provide an error message.

Description

This function sends a request to database to retrieve information about available Generative AI models.

Detail

The function utilizes the GenAI database API to fetch the latest information about available Generative AI models. The retrieved data includes details about different models offered by various service providers.

Example 
models = available.models()

connect.genai

Set up and connect to a Generative AI model

Usage
connect.genai(provider,
              model, 
              version, 
              api, 
              proxy = FALSE)
Arguments

  • provider: A character string representing the Generative AI service provider (e.g., "google", "openai").

  • model: A character string representing the specific model offered by the selected provider (e.g., "gpt-3.5-turbo", "gemini-pro").

  • version: A character string representing the version of the chosen model.

  • api: A character string representing the API key required for accessing the model.

  • proxy: A boolean value indicating whether to use a proxy for accessing the API URL (default is FALSE). If your local internet cannot access the APIs, set this parameter to TRUE.

Value

If successful, the function returns a vector containing information about the chosen Generative AI service provider, model, version, API key, and proxy status. If the API response indicates an error, the function halts execution and provides an error message.

Description

This function establishes a connection to a Generative AI model by providing essential parameters such as the service provider, model, version, API key, and proxy status.

Detail

It is crucial to provide accurate and valid information for each parameter to ensure a successful connection to the Generative AI model. If any of the provided parameters are incorrect, the function will respond with an error message based on the information received from the API. Use the function available.models to see all supported Generative AI models.

Example 
models = available.models()
models

google.model = connect.genai("google",
                             models$google$model[1],
                             models$google$version[1],
                             "API_KEY",
                             FALSE)
google.model

openai.model = connect.genai("openai",
                             models$openai$model[1],
                             models$openai$version[1],
                             "API_KEY",
                             FALSE)
openai.model

Text Generation

The "Text Generation" section comprises functions for generating text using Generative AI models. The txt function allows users to generate text based on a provided prompt and model parameters, with options to adjust creativity using the temperature parameter. Additionally, specialized functions like txt.explain.code, txt.fix.grammar, txt.image, and txt.optimize.code cater to specific text generation needs, such as explaining code, fixing grammar issues, incorporating images into text generation, and optimizing code based on a goal. Each function establishes a connection to a Generative AI model and returns the generated text, emphasizing the importance of accurate parameter input for successful text generation. Users can explore supported models using the available.models function.

txt

Generate text using the Generative AI model based on a prompt

Usage
txt(model.parameter, 
    temperature, 
    prompt)
Arguments

  • model.parameter: A character vector containing the Generative AI service provider, corresponding model, version, API key, and proxy status.

  • temperature: A numeric value. A higher value results in more creative responses, while a lower value produces more straightforward text.

  • prompt: A character string representing the query for text generation.

Value

If successful, a character string generated by the Generative AI model based on the provided query and parameters. If the API response indicates an error, the function halts execution and provides an error message.

Description

This function establishes a connection to a Generative AI model by providing essential parameters and generates a response based on the prompt.

Detail

Providing accurate and valid information for each parameter is crucial to ensure successful text generation by the Generative AI model. If any of the provided parameters is incorrect, the function will respond with an error message based on the information received from the API. Use the function available.models to see all supported Generative AI models.

Example 
models = available.models()

google.model = connect.genai("google",
                             models$google$model[1],
                             models$google$version[1],
                             "API_KEY",
                             FALSE)

Chat Generation

The "Chat Generation" section offers a set of functions for interacting with Generative AI models in a conversational manner. The primary function, chat, initiates a conversation by establishing a connection to a Generative AI model using parameters like service provider, model, version, API key, and proxy status. It generates a response based on the provided prompt and stores it in the chat history list. Other functions, such as chat.convert, chat.edit, chat.recent, chat.save, and chat.setup, provide additional capabilities like converting chat history between models, editing messages, retrieving the latest message, saving the chat history in different formats, and setting up an empty chat history list. Each function emphasizes the importance of accurate parameter input for successful interaction with Generative AI models, and users can explore supported models using the available.models function.

chat

Initiate a conversation with a Generative AI model using a prompt

Usage
chat(model.parameter, 
     temperature, 
     history, 
     prompt)
Arguments

  • model.parameter: A character vector containing information about the Generative AI service provider, corresponding model, version, API key, and proxy status.

  • temperature: A numeric value. Higher values yield more creative responses, while lower values result in more straightforward text.

  • history: A list containing all messages exchanged between the user and the model in a chat session.

  • prompt: A character string representing the query for text generation.

Value

If successful, the full chat history, including the most recent response, will be returned. If the API response indicates an error, the function halts execution and provides an error message.

Description

This function establishes a connection to a Generative AI model by providing essential parameters. It generates a chat response based on the provided prompt and stores it in the history list.

Detail

Providing accurate and valid information for each parameter is crucial for successful text generation by the Generative AI model. This function modifies the chat history. If any parameter is incorrect, the function responds with an error message based on the API feedback. To view all supported Generative AI models, use the function available.models. Additionally, utilize chat.setup to set up a chat history. To retrieve the most recent response generated by the model, use the function chat.recent.

Example 
models = available.models()

google.model = connect.genai("google",
                             models$google$model[1],
                             models$google$version[1],
                             "API_KEY",
                             FALSE)

openai.model = connect.genai("openai",
                             models$openai$model[1],
                             models$openai$version[1],
                             "API_KEY",
                             FALSE)

google.history = chat.setup(google.model)
openai.history = chat.setup(openai.model)

temperature = 0.9
prompt = "Write a story about a magic backpack in about 100 words."

google.history = chat(google.model,
                      temperature,
                      google.history,
                      prompt)
cat(chat.recent(google.model, google.history))

openai.history = chat(openai.model,
                      temperature,
                      openai.history,
                      prompt)
cat(chat.recent(openai.model, openai.history))

prompt = "What is the word count of the story you just wrote?"

google.history = chat(google.model,
                      temperature,
                      google.history,
                      prompt)
cat(chat.recent(google.model, google.history))

openai.history = chat(openai.model,
                      temperature,
                      openai.history,
                      prompt)
cat(chat.recent(openai.model, openai.history))

chat.convert

Convert chat history from one Generative AI model to another

Usage
chat.convert(source.model.parameter, 
             source.history, 
             target.model.parameter)
Arguments

  • source.model.parameter: A character vector containing essential information about the source Generative AI service provider, model, version, API key, and proxy status.

  • source.history: A list containing all messages exchanged between the user and the model in a chat session.

  • target.model.parameter: A character vector containing essential information about the target Generative AI service provider, model, version, API key, and proxy status.

Value

If the conversion is successful, the function returns the transformed chat history in list format. In case of incorrect input formats, the function halts execution and provides an error message.

Description

This function transforms the chat history from one Generative AI model to another, ensuring compatibility and usability across different models.

Detail

Providing accurate and valid information for each parameter is crucial for successful text generation by the Generative AI model. This function modifies the chat history accordingly. Incorrect parameters will result in an error message based on the API feedback. To explore supported Generative AI models, use the function available.models. Additionally, utilize chat.setup to set up a chat history.

Example

                      temperature,
                      openai.history,
                      prompt)
cat(chat.recent(openai.model, openai.history))

google.history = chat(google.model,
                      temperature,
                      google.history,
                      prompt)

openai.history = chat(openai.model,
                      temperature,
                      openai.history,
                      prompt)
cat(chat.recent(openai.model, openai.history))

converted.google.history = chat.convert(openai.model,
                                        openai.history,
                                        google.model)

converted.openai.history = chat.convert(google.model,

                                        openai.model)

                                temperature,

chat.setup

Generate an empty chat history list for a Generative AI model

Usage
chat.setup(model.parameter)
Arguments

  • model.parameter: A character vector containing the Generative AI service provider, corresponding model, version, API key, and proxy status.

Value

If successful, a chat history list is generated for the Generative AI model.

Description

This function establishes a connection to a Generative AI model by providing essential parameters and generates an empty chat history list for the model.

Detail

Providing accurate and valid information for each parameter is crucial to ensure successful text generation by the Generative AI model. The chat history will be modified by this function. If any of the provided parameters is incorrect, the function will respond with an error message based on the information received from the API. Use the function available.models to see all supported Generative AI models.

Example 
models = available.models()

google.model = connect.genai("google",
                             models$google$model[1],
                             models$google$version[1],
                             "API_KEY",
                             FALSE)

openai.model = connect.genai("openai",
                             models$openai$model[1],
                             models$openai$version[1],
                             "API_KEY",
                             FALSE)

google.history = chat.setup(google.model)
openai.history = chat.setup(openai.model)

temperature = 0.9
prompt = "Write a story about a magic backpack in about 100 words."

google.history = chat(google.model,
                      temperature,
                      google.history,
                      prompt)
cat(chat.recent(google.model, google.history))

openai.history = chat(openai.model,
                      temperature,
                      openai.history,
                      prompt)
cat(chat.recent(openai.model, openai.history))
